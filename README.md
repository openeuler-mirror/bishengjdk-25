# bishengjdk-25

## 介绍

毕昇JDK是华为内部OpenJDK定制版Huawei JDK的开源版本，是一个高性能、可用于生产环境的OpenJDK发行版。Huawei JDK运行在华为内部500多个产品上，积累了大量使用场景和java开发者反馈的问题和诉求，解决了业务实际运行中遇到的多个问题，并在ARM架构上进行了性能优化，毕昇JDK运行在大数据等场景下可以获得更好的性能。毕昇JDK 25目前支持Linux/AArch64、Linux/x86_64平台。毕昇JDK同时是OpenJDK的下游，现在和未来也会持续稳定为OpenJDK社区做出贡献。

## 平台支持

毕昇JDK 当前支持 `Linux/AArch64、Linux/x86_64` 平台。
